# DexCompose Project Page

This repository contains the static project page for:

**DexCompose: Reusing Dexterous Policies for Multi-Task Manipulation with a Single Hand**

The page is served from `index.html` and uses local static assets under `static/`.

## Release Links

- Project page: https://dexcompose.github.io/DexCompose-Page/
- Code: https://github.com/DexCompose/DexCompose
- Paper PDF: `static/papers/dexcompose_arxiv.pdf`
- arXiv: replace the disabled arXiv button in `index.html` with the arXiv `abs` URL after the public entry is available.

## Preview

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
